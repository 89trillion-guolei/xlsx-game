# 解析游戏配置excel

# 1.基本格式 
       第一行个字段说明，中文或英文
       第二行英文，字段KEY
       第三行开始，数据（第一列必填，否则会跳过）

# 2.数据格式
       浮点数含 “.”        例：22.02
       数组“|”相隔         例:  10|20|30
       整数  数字            例：20
       其他默认字符串
       
默认文件名game.xlsx, 放置同一目录下。执行xlsx.bat， 编辑xlsx.bat可以配置解析N个文档<br>
命令行执行:  xlsx -f  game.xlsx （xx为文件路径）<br>
生成JSON的文件名同Excel文件名
